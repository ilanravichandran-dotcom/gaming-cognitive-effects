# Video Games and the Brain - Educational Website

An informative website exploring how video games affect the brain, covering both positive effects and potential concerns based on scientific research.

## Features

- **Comprehensive Information** - Covers positive effects, concerns, research findings, and healthy gaming tips
- **Modern Design** - Clean, colorful, and engaging interface
- **Responsive Layout** - Works on desktop, tablet, and mobile devices
- **Smooth Animations** - Cards animate as you scroll for better user experience
- **Research-Based Content** - Information grounded in neuroscience and psychology research

## Sections

1. **Home** - Introduction to the topic
2. **Benefits** - Positive cognitive effects of gaming
3. **Concerns** - Potential negative impacts
4. **Research** - Scientific findings about brain structure and function
5. **Tips** - Healthy gaming practices

## Getting Started

1. Open `index.html` in your web browser
2. Or run a local server:
   ```bash
   python3 -m http.server 8000
   ```
   Then visit `http://localhost:8000`

## Customization

You can easily customize:
- Content and research findings
- Colors in `styles.css`
- Add more sections or information
- Include images or videos
- Add interactive elements

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)

## Educational Use

This website is designed for educational purposes and provides general information about video games and brain function. It is not a substitute for professional medical or psychological advice.
